## How to build
g++ -std=c++17 Assignment6.cpp -o BankAccount

## How to run
./BankAccount   # or BankAccount.exe on Windows

#	Option
1	Balance
2	Make a Deposit
3	Make a Withdrawal
4	Deposit History
5	Withdrawal History
6	All Deposits & Withdrawals
7	Exit (print starting & final balance)
